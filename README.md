## 🚀 What neurogent does

Use it when you want to:

- build AI agents from a simple workflow
- package an agent into a shareable form
- deploy an agent with a clean command line flow
- keep your setup aligned with A2A rules
- add a security layer during agent use

## 🧪 Troubleshooting

If the app does not open:

- check that the file finished downloading
- extract the ZIP folder if there is one
- try running the file as admin
- make sure Windows did not block the file

If the app opens but closes fast:

- open it again from the same folder
- keep the project files in place
- check whether the app needs a config file
- look for a log file in the app folder

If Windows says it cannot find a file:

- confirm the folder was not moved
- re-download the file
- extract it again to a simple folder like Downloads\neurogent

If the CLI window flashes and closes:

- open Command Prompt first
- run the file from inside that window
- keep the window open so you can read the message
